# Student-Management-System--Python+pandas+Matplotlib

A menus-driven student management system 

this project allows manging student records and performing basic data


## 🚀 Features

- Add new students  
- Show all students  
- Search student by name  
- Delete student by roll number  
- Update student marks  
- Grade calculation  
- Data analysis using graphs:
  - Bar chart of marks
  - Histogram of marks distribution 

## 🛠 Technologies Used

- Python  
- Pandas  
- Matplotlib
