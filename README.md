# Student-Management-System--Python
A console-based student management system in Python

A console-based Student Management System built using Python and file handling.
This project allows users to manage student records stored in a text file.

# Features
- Add new student
- Display all students
- Search student by name
- Update student marks
- Delete student record
- Calculate statistics:
  - Total students
  - Average marks
  - Highest & lowest scorers
  - Grade calculation (A, B, C, D, Fail)

# Technologies Used
- Python 3
- File Handling
- Exception Handling
- Object-Oriented Programming (OOP)
